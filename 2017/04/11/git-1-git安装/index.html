安装TortoiseGit

使用原生的git软件查看文件的diff信息是很痛苦的一件事，各种++–会把人看到崩溃，自带的gui功能又很简陋，因此推荐安装TortoiseGit查看diff。

事实上，TortoiseGit是一个很全面的git管理软件（Tortoise还有类似的SVN管理软件），使用它我们可以完成所有git操作。不过我个人认为除了diff外，git原生软件在其他功能上实现的都不错，使用原生的软件还要更为顺手一些。
